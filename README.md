Github Cloner
=============

An interactive command line tool to list and clone your Github repositories.

Dependencies
------------
 - node
 - ncurses

Installation
------------

###NPM

    (sudo) npm install -g github-cloner

###Sources
    git clone https://github.com/floriancargoet/github-cloner.git
    npm install
